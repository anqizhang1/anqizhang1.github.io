# Images Folder

Place your portrait photo here, named `anqi.jpg`.

A 4:5 aspect ratio (such as 800 × 1000 pixels) fits the celadon frame in the About section best. Other ratios will be cropped to fit using CSS `object-fit: cover`.

After adding your photo, update the comment in `index.html` (in the About section) to reference it as an `<img>` tag. Full instructions are in the main `README.md`.
